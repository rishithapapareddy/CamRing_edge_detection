# CamRing_edge_detection
Implementation of camring detection using image processing techniques to detect edges in images

Objective
To detect and highlight circular edges in images using edge detection techniques.

Technologies Used
Python
OpenCV
NumPy
Matplotlib

Methodology
Load input image
Convert image to grayscale
Apply Gaussian Blur for noise reduction
Perform edge detection using Canny Edge Detection
Detect circular shapes using Hough Circle Transform
Display the detected ring edges

Results
Successfully detected circular edges in sample images.
The system highlights detected rings accurately.
